The conclusion

The framework is getting smoother and smoother in making the Angular platform best. The main reason for many of us to upgrade to new versions of angular is to get a performance boost.

- Did it perform to your expectations?
Yes, it is as per our expectations.

- Was it difficult to install or configure?
No. It was easy to install and configure. However we had faced some issues due to mismatch of NodeJS (updated) and Angular version compatibility. But we resolved it quickly, as there are many communities are available for Angular.

- Was it easy to create the page(s) with it?
Yes, it is easy to create the pages with it. Also, routing feature is awesome to link the pages in the single page application (SPA).

- Was the learning curve steep compared with regular HTML/CSS/JavaScript/PHP?
Yes, it is bit complex as compare to regular HTML/CSS/JavaScript but once we learned it then developed becomes smooth and fast.

- Would you have done things differently in retrospect?
Yes, we have used Angular and their features like routing, SCSS for CSS

- Did you regret your choice of framework?
No. We are happy with our selection for Angular.
